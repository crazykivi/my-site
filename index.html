Сайт ПКФ "ХозКомплект" был разработан с целью создания удобного интернет-ресурса для оптовых клиентов. Платформа позволяет пользователям легко выбирать товары из широкого каталога и оформлять заказы с помощью сервиса Yandex SMTP, который используется для обработки заявок.

В будущем система будет интегрирована напрямую с базой данных MySQL, что обеспечит прямое взаимодействие с 1С для автоматизации процессов.

Обратите внимание, что сайт пока не имеет привязанного домена, и SSL-сертификаты не установлены!

В рамках этого проекта был внедрён эмулятор ТСД, который имитирует работу терминала для ввода товаров. Перейти к проекту


Это мой личный сайт, на котором я планирую размещать информацию о своих проектах. На данный момент сайт не использует базы данных, так как для его работы в этом нет необходимости.

Сайт развернут в контейнере Docker с использованием веб-сервера Nginx. Все изменения в проекте осуществляются через систему контроля версий GitHub. Внесенные изменения автоматически загружаются в репозиторий на GitHub.

Для автоматизации процесса развертывания используется GitHub Actions. Это позволяет контейнеру с новым кодом автоматически разворачиваться на сервере каждый раз при обновлении репозитория. Таким образом, любые изменения, отправленные в репозиторий на GitHub, автоматически обновляют сайт без необходимости вручную развертывать изменения.

Контейнер получает настройки Nginx с виртуальной машины, однако в дальнейшем планируется оптимизация этого процесса.

Обратите внимание, что сайт пока не имеет привязанного домена, и SSL-сертификаты не установлены!

О сайте

Сайт развернут в контейнере Docker с использованием веб-сервера Nginx. Все изменения на сайте выполняются через GitHub. Все изменения пушатся в этот репозиторий.

Для автоматизации развертывания используется GitHub Actions, что позволяет автоматически разворачивать контейнер с новым кодом на сервере при каждом изменении в репозитории. Таким образом, при каждом пуше на GitHub, соответствующие изменения автоматически внедряются в систему и обновляют сайт без необходимости ручного развертывания.
Контейнер берет настройки Nginx с виртуальной машины (в дальнейшем планируется изменить данный процесс).

Все исходные коды лежат на данном репозитории, инструкция по развертыванию будет написана позже.
